User's manual

Count down timer for your nap :) Alarm will go off after you turn off the screen and have kept it off for a certain amount of time.

Notes

The alarms might be disabled by task killer applications. Users of those applications should make sure that the alarms go off after killing Siesta Watch manually or add Siesta Watch to the ignore list before serious use.

Usage

Setting Alarm

  • Set the time duration you want to Sleep for. Alarm will go off after the screen is kept off for this amount of time.
  • Optionally, set Time Limit and check the box. Alarm will go off when the time comes.
  • You can choose to use Vibration with the audible alarm.
  • Finally, push the Set button to set the alarm.

Waking up

When alarm goes off, turn the screen on (or off) to silence the alarm. If you turn on the screen, you also have to unlock it to stop the alarm.

Canceling Alarm

When you want to cancel the alarm, start the application and push the Cancel button.
